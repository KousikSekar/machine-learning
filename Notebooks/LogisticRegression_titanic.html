Logistic Regression¶

Logistic Regression is a classification Algorithm , meaning we use to split the data based on some qualities like eye color (brown , grey , blue) , severity (mild , moderate , severe ) etc¶

Linear regresssion are used to predict the quantitative data whereas the logistic regressions are used to classify the qualitative data .¶

The graph above gives us a comparison of the linear model (left) and logistic model (right)¶

To split up the predicted values we would require our values to be as a probability , so in logistic model we convert our linear regression into probability funtion by substituting the regression equation into sigmoid function¶

Many functions can bring our linear eqn values between 0 to 1 , in logistic regression we use the sigmoid function¶

So the logistic regression classifies based upon the probability value¶

Let's predict whether a passengar can or will survive or not in Titanic¶

#Let's fill the null age values 
def fillAge(param):
    Age = param[0]
    Pclass = param[1]
    if(pd.isnull(Age)):
        if(Pclass == 1):
            return 37
        elif(Pclass == 2):
            return 29
        else:
            return 25
    else:
        return Age
